IRS Connection
TDS API — pull client transcripts directly from the IRS
How this works: Using Romeo's IRS e-Services API credentials (A2A OAuth flow), this connects to the IRS Transcript Delivery System. The system generates two signed JWTs using Romeo's private key, exchanges them for an access token, then requests the client's transcript via CAF authority — exactly as the IRS API documentation specifies. Credentials must be set in the Credentials tab first.

IRS API Credentials
How to securely configure Romeo's IRS e-Services credentials
⚠️ Never enter credentials in this UI. Anything typed in a browser can be intercepted or logged. All IRS credentials must be added directly in Netlify — never through any form.
✅ How to add credentials securely
Add these directly in Netlify — takes 2 minutes.
  1. Go to app.netlify.com → your site
  2. Click Site configuration → Environment variables
  3. Click Add a variable for each one below
  4. After adding all variables, push any commit to redeploy
  5. Come back here and click Check Status to verify
🔍 Current Status
Click "Check Status" to verify which credentials are configured.
📋 Variables to add in Netlify
Copy each variable name exactly as shown. Values come from your IRS e-Services account.
Variable Name What it is How to get it
IRS_API_CLIENT_ID Your IRS API application ID irs.gov/e-services → Access API Client ID → Application Details page
IRS_ESERVICES_USERNAME Romeo's IRS e-Services login username The username Romeo uses to log into irs.gov/e-services
IRS_CAF_NUMBER 9-digit CAF number (no hyphens) Call IRS Practitioner Priority Service: 866-860-4259
IRS_JWK_KID Key ID from your JWK registration From the JWK file you uploaded during IRS API registration (usually "key1")
IRS_PRIVATE_KEY_PEM RSA-2048 private key in PEM format Generated when you created your key pair for IRS registration. Starts with -----BEGIN PRIVATE KEY-----
🔑 One-time setup steps
Step 1 — Register for IRS API access Log into irs.gov/e-services → navigate to Access API Client ID → complete the application form. You'll need to generate an RSA-2048 key pair and upload the public key as a JWK file with x5c and x5t parameters. Your Client ID appears on the Application Details page once approved.
Step 2 — Get your CAF number Call the IRS Practitioner Priority Service at 866-860-4259. This is the 9-digit number assigned to Romeo as an authorized representative. It's required in every transcript request.
Step 3 — Grant A2A consent (one-time) Romeo must log into the IRS Consent App at la.www4.irs.gov/esrv/consent/ and authorize the Client ID for A2A access. This is a one-time step that allows the system to pull transcripts without Romeo being present.
Step 4 — Test the connection After adding all variables and redeploying, go to the IRS Connection tab and click Test Connection. A green success message confirms everything is working.
